About ToDo

**ToDo** is a comprehensive, full-stack To-Do list web application designed for personal productivity and built to showcase modern development skills.

Technical Foundation

This application is powered by **Flask (Python)** on the backend, using **SQLAlchemy** for efficient database management (ORM), and **Jinja2** for dynamic server-side templating.

Security and user experience were core priorities, featuring secure user authentication (signup/login) implemented with **Flask-Login** and robust password hashing via **Bcrypt**.

Key Features

Users can efficiently manage their workflows with features like:

  • **User Isolation:** Each user has private, secure access to their own tasks.
  • **Dynamic Task Filtering:** Tasks are automatically categorized and filterable by **Overdue**, **Due Today**, and **Pending** status based on the due date.
  • **Real-Time Status Toggle:** Quick links to mark tasks complete or incomplete.
